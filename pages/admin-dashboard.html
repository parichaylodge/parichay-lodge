Staff Accounts
👤

Manage Staff via Firebase Console

Create and manage staff users directly in Firebase Authentication, then assign roles in Firestore.

Open Firebase Console ↗
Setup Guide
1
Create user in Firebase

Firebase Console → Authentication → Users → Add User → Enter email & password

2
Copy the UID

After creating, click the user row and copy the User UID (long string)

3
Add to Firestore

Firestore → users collection → Add document → ID = UID → field: role: "staff" or role: "admin"
